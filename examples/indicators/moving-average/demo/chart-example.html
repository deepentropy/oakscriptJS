OakScriptJS - Moving Average Indicator

Interactive example showing how to use OakScriptJS with Lightweight Charts v5. Adjust the parameters below to see real-time updates.

How it works:

This example demonstrates the OakScriptJS library calculating moving averages directly from chart data. The calculation happens in the browser using the calculateMovingAverage() function from examples/indicators/moving-average/index.ts.

Key features:

  • Real-time updates when changing parameters
  • Support for multiple MA types (SMA, EMA, WMA)
  • Configurable source data (close, open, high, low, etc.)
  • Optional offset and smoothing
  • Only 49 lines of indicator code vs 355 lines in official TradingView implementation
