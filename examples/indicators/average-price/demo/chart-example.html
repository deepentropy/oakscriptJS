OakScriptJS - Average Price Indicator

Average Price calculates (Open + High + Low + Close) / 4 for each bar. Adjust the offset to shift the line forward or backward.

How it works:

This example demonstrates the Average Price indicator using OakScriptJS. The calculation is simply: (O + H + L + C) / 4
